Projects

SimRNA

SimRNA is a software for RNA folding simulation and RNA 3D structure prediction using coarse-grained representation, knowledge-based potential, and Monte Carlo sampling method.
My duties in this project were to add additional features to the program, such as new types of restraints, comparative modeling with more than one template, modified nucleotides, parallel isothermal calculations, compatibility with mmCIF files, and to generate restraints for noncanonical base pairs using deep learning.

SimDNA

SimDNA is a software for DNA folding simulation and DNA 3D structure prediction using coarse-grained representation, knowledge-based potential, and Monte Carlo sampling method.
My duties My duties in this project included developing the software from the ground up, based on the latest version of SimRNA. I prepared the dataset, calculated the scoring function, wrote the code, and conducted a small benchmark for the beta version of the software.
